--- title: Add all the natural numbers below one thousand that are multiples of 3 or 5 description: Add all the natural numbers below one thousand that are multiples of 3 or 5 created: !!timestamp '2011-09-03 18:00:00' keywords: project euler, tags: - r - python - scala --- Problem ------- If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23. Find the sum of all the multiples of 3 or 5 below 1000. Python ------ ### Solution 1 Comprehensions with conditions {% syntax python %} sum([i for i in range(1,1000) if not i%3 or not i%5]) {% endsyntax %} ### Solution 2 Comprehensions with conditions are similar to filter {% syntax python %} sum(filter((lambda i: i%3 == 0 or i%5 == 0), range(1,1000))) {% endsyntax %} Both solutions looks elegant, but it seems first solution is faster especially when we increase the upper range from 1000 to 100000 difference is more than 4 fold. {% syntax python %} >>> import cProfile >>> cProfile.run('sum([i for i in range(1,100000) if not i%3 or not i%5])') 4 function calls in 0.054 CPU seconds Ordered by: standard name ncalls tottime percall cumtime percall filename:lineno(function) 1 0.047 0.047 0.054 0.054 :1() 1 0.000 0.000 0.000 0.000 {method 'disable' of '_lsprof.Profiler' objects} 1 0.006 0.006 0.006 0.006 {range} 1 0.001 0.001 0.001 0.001 {sum} >>> cProfile.run('sum(filter((lambda i: i%3 == 0 or i%5 == 0), range(1,100000)))') 100004 function calls in 0.288 CPU seconds Ordered by: standard name ncalls tottime percall cumtime percall filename:lineno(function) 99999 0.141 0.000 0.141 0.000 :1() 1 0.005 0.005 0.288 0.288 :1() 1 0.135 0.135 0.276 0.276 {filter} 1 0.000 0.000 0.000 0.000 {method 'disable' of '_lsprof.Profiler' objects} 1 0.005 0.005 0.005 0.005 {range} 1 0.002 0.002 0.002 0.002 {sum} {% endsyntax %} R Lang ------ {% syntax r %} x <- 1:999 sum(x[x %% 3 ==0 | x %% 5 == 0]) {% endsyntax %} R took nearly 0.018 seconds for 100000 {% syntax r %} x <- 1:100000 system.time(sum(as.numeric(x[x %% 3 ==0 | x %% 5 == 0]))) user system elapsed 0.020 0.000 0.018 {% endsyntax %} Scala ----- TO DO
